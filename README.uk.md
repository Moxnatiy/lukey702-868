# Lukey 702 / 868 — відкрита прошивка

Реверс-інжиніринг і переписана відкрита прошивка для паяльної станції **Lukey 702**
(на платі маркування **Lukey 868**), SMD rework. У станції **два мікроконтролери**
(обидва ATmega8): один керує **феном (гарячим повітрям)**, інший — **паяльником**.

> 🇬🇧 [Read in English](README.md)

Оригінальна прошивка була заблокована від читання (read-protected) і працювала некоректно,
тож проєкт відтворює прошивку кожного МК з нуля — розводку пінів відновлено зі схеми та
референсної прошивки, далі чиста задокументована реалізація на C із меню калібрування
прямо на станції.

## Структура репозиторію

| Тека | МК | Стан |
|------|-----|------|
| [`hot-air/`](hot-air/) | Фен — ATmega8L | ✅ працює: повний PID, меню CAL на станції, прошито й протестовано |
| [`solder-iron/`](solder-iron/) | Паяльник — ATmega8 | 🚧 заплановано |

Кожен підпроєкт самодостатній (код, документація, Makefile, конфіг VS Code).

## Залізо

- **МК:** ATmega8L-8PU, внутрішній RC 8 МГц (без зовнішнього кварцу).
- **Плата:** Scorpio «LUKEY868(702) SMD REWORK».
- **Програматор:** USBASP (ISP). Прошивка через `avrdude`.
- **Тулчейн:** `avr-gcc` (зібрано тулчейном AVR від PlatformIO, gcc 7.3.0).

## Швидкий старт (фен)

```bash
cd hot-air
make            # зібрати -> fan.hex
make flash      # chip-erase + прошивка через USBASP
```

Деталі, карта пінів і меню калібрування — у [`hot-air/README.md`](hot-air/README.md).
